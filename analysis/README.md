# analysis/
Código Python para procesar datos GTFS y calcular indicadores de red.

- `notebooks/` — Exploración interactiva con Jupyter
- `scripts/` — Scripts Python listos para producción, llamados desde `pipeline/run_pipeline.py`

Herramientas principales: Apimetro (client HTTP en Go/Python) y VFTModel (indicadores DI, B(v), ΔE).
